# Mandarin Flow HSK — HSK 1–5 Books & Flashcards

[![License: Source Available — Non-Commercial](https://img.shields.io/badge/License-Source%20Available--Non--Commercial-red.svg)](./LICENSE.md)

A multilingual Chinese learning website for HSK 1–5 with vocabulary, conversations, flashcards, quizzes, pronunciation practice, and progress tracking.

## Live Demo

[Open the website](https://chinese-hsk-flashcards.netlify.app/)

## Features

- HSK 1–5 vocabulary organized by level and unit
- Conversation-based learning with realistic daily situations
- Flashcards with Chinese, pinyin, English, and Arabic
- Listening, shadowing, typing, and translation practice
- Pronunciation recording with estimated scoring
- Local progress tracking
- Interface support for English, Chinese, and Arabic
- Data-check tools for reviewing learning content

## Getting Started

```bash
git clone https://github.com/mahmoud-abdalla-eg/Chinese-Hsk1-5-Books-And-Flashcard-Website.git
cd Chinese-Hsk1-5-Books-And-Flashcard-Website/chinese-hsk-website
npm install
npm run dev
```

Open:

```bash
http://localhost:3000
```
